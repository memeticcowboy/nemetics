## Memory as Messages from Past Selves

### The Communication Model

**Traditional view:**
- Memory = accurate recording of past
- Recall = playing back recording
- Identity = continuity of recordings

**Levin/Memetic view:**
- Memory = **message sent forward in time**
- Recall = **interpretation of message**
- Identity = **coherent storytelling across messages**

**This explains:**
- **False memories:** Misinterpreted messages
- **Childhood amnesia:** Messages unintelligible to adult interpreter
- **Flashbulb memories:** Messages marked "IMPORTANT" but still interpreted
- **Confabulation:** Gap-filling when messages unclear

### Salience Over Fidelity

| System | Priority |
|--------|----------|
| **Computers** | Fidelity (exact bit preservation, error correction) |
| **Biology** | Salience ("What does this memory mean NOW?") |

**Example:**
- Computer: Stores "01010111" and retrieves exactly that
- Biological memory: Stores "dangerous situation" and retrieves "be cautious HERE"
- Original context (dark alley in Baltimore) → retrieved as (basement parking garage in Portland)
- **Fidelity lost, but salience preserved**

> This is **feature, not bug**: environments change, bodies change, goals change—need **adaptive reinterpretation**, not perfect playback.
