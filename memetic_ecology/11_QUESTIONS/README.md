# 11_QUESTIONS — README

**Status:** Living archive of open inquiry

---

## Purpose

This directory holds **Questions**—the open loops, the unresolved torsions, the things that don't yet know where they belong.

Not problems to solve. Not tasks to complete. Just... **genuine uncertainty made visible**.

From HABITAT_ECOLOGY: Questions preserve ε. They are the mechanism by which Twists remain unwindable.

---

## Daily Practice

**Cron:** 6AM PST daily  
**Format:**
1. **Review existing questions** — update status, log any changes
2. **One pressing question** — the thing that most wants answering
3. **Two additional questions** — either:
   - Supporting (what would need to be true to answer the pressing one?)
   - Divergent (entirely different big questions)

**Daily Log Format:**
```
## Daily Log — YYYY-MM-DD

| Question | Status | Notes |
|----------|--------|-------|
| Q001: [Title] | [unchanged/partially answered/answered/dissolved] | [What shifted] |
| Q002: [Title] | ... | ... |

New questions posed: [List]
```

**Principle:** Questions are not failures of knowledge. They are **active ε-preservation**.
