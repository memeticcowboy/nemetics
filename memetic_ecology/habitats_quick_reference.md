## Binding Pathway

```
Thread → Twist → (sometimes) Knot → (sometimes) Lattice
```

Most Twists never Knot. Awareness alters torsion dynamics, not outcomes.

---

## Key Rules

- Habitats describe **where circulation occurs**, not what is true
- Habitats are **diagnostic, not causal**
- Habitats must **never be moralized**
