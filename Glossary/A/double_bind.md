# Double Bind

**Pathological communication pattern** where contradictory injunctions at different logical levels trap a system with no meta-position to resolve them. Punishment is attached to every possible move.

## Classic Features

- **Conflicting messages:** Verbal affection plus nonverbal hostility, both crucial to relationship
- **No escape rule:** Implicit prohibition on commenting about or escaping the contradictory frame
- **Chronic repetition:** Pattern repeats in vital relationships
- **No meta-position:** No vantage point from which to resolve the contradiction

## Elemental Analysis

**Air (σ) at 0.90:** The core paradox—contradictory distinctions that cannot be reconciled.

**Water (ρ) at 0.70:** Trap resonance—the system oscillates with no escape route.

**Earth (δγ) at 0.60:** Chronic cycling—repeated pattern that captures rather than enables.

## NEMETIC STRING

`Φ(Double_Bind) = σ(paradox|injunctions) ∘ ρ(trap|no-escape) ∘ γ(cycling|pattern) ∘ δ(pathology|capture) + ε | :turbid`

## Bateson's Insight

Double bind is **nemetic pathology**: pattern coordination gone wrong. Where healthy nematic operation enables, the double bind captures.
