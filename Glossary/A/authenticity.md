# Authenticity

**Expression-of-place and memory as active, reconstructed through cycles.**

Not preservation but reconstruction. Emerges through δγ-cycling (metabolism). Place-bound but not place-fixed. Requires release for regeneration.

**Principle:** Authenticity is not a state to achieve but a process to participate in.

**Diagnostic:** Can this authenticity be revised without identity rupture?

**Related:** Metabolism, Stewardship, Permeability
