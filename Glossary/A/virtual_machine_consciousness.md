# Virtual Machine Consciousness

**Brain hosts layered software processes**—sensory, linguistic, narrative—with 'self' as one such executable. Consciousness as virtual machine running on neural hardware.

## Layered Architecture

| Layer | Function |
|-------|----------|
| **Sensory** | Raw signal processing |
| **Linguistic** | Symbolic manipulation |
| **Narrative** | Self as story |

Each layer emerges from prior. 'Self' is software process, not hardware.

## Key Features

- **Memory:** Not retrieval but generative potential—on-demand recomposition
- **Diversity:** Aphantasia, absence of inner speech show linguistic cognition not universal
- **Jailbreakable:** Mind as operating system that can be hacked, reprogrammed, captured
