# Fractal Depth

**Recursive, nested emergence that expands dimensionality rather than regresses.**

Not vertical (higher/lower) but topological. Nested bow-ties: each ✶ becomes substrate for next. Lossy iteration: ε ≠ 0 ensures new information enters.

**Formula:** `Previous ✶ → New σ → New ρ → New λ → New ✶`

**Diagnostic:** Does each iteration add information or merely repeat?

**Related:** ✶-State, Metabolism, Tertiary Emergence
