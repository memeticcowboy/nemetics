# False Memory

**Vivid, emotionally compelling memories constructed from imagination, suggestion, culture.**

Not just errors — constructed experiences. Earth (δγ) integration processes hijacked; Water (ρ) felt-quality of "realness"; Air (σ) source monitoring fails. Confabulation fills gaps with plausible fabrications.

**Principle:** "Memory is reconstruction, not playback."

**Diagnostic:** Can the memory be corroborated? Does it align with timeline? Is there external evidence?

**Related:** Source Monitoring Error, Absorption, AI Hallucination
