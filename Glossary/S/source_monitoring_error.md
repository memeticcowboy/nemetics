# Source Monitoring Error

**Confusing internal imagination with external perception.**

"Did I imagine that or did it happen?" Air (σ) distinction failure between memory sources. Underlies alien abduction beliefs, false memories, AI hallucination recall. Reality monitoring degrades.

**Principle:** "The mind cannot always distinguish between dreamed and lived."

**Diagnostic:** Can you trace the source of this memory? Is there external corroboration?

**Related:** False Memory, AI Psychosis, Absorption
